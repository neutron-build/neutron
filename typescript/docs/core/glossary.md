# Neutron Glossary

## Neutron

Umbrella platform and framework brand.

## Neutron TypeScript

TypeScript implementation of the Neutron framework.

## Neutron Rust

Rust implementation of the Neutron framework.

## Neutron Zig

Zig implementation of the Neutron framework.

## Neutron Mojo

Mojo implementation of the Neutron framework.

## Nucleus

Shared core plane/services layer used by the broader Neutron ecosystem.

## Layering Rule

Keep platform (`Neutron`), implementation (`Neutron <Language>`), and subsystem (`Nucleus`) as separate names.

Do not combine them into one compound name such as `neutron-typescript-mojo-nucleus`.

## Framework Contract

Language-agnostic behavioral spec all implementations should conform to.

## Implementation Docs

Language-specific setup/runtime/tooling docs for each Neutron implementation.
