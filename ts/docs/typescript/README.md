# Neutron TypeScript Docs

This is the Neutron TypeScript implementation.

Neutron TypeScript is a full framework implementation intended to be used independently.

It should be documented as production-grade and complete in scope (not as a thin adapter layer).

## Coverage

1. Installation and project bootstrap
2. Framework APIs and patterns
3. Build/dev/preview/deploy workflows
4. Performance, caching, and security guidance
5. Migration and release notes
